inst_inverter Inverter
Instantiation

The Inverter and the AXI4Stream_InverterWrapper shares the same generics and most of the ports. The only difference is that we have subdivided the 32 bits of s00_axis_tdata in pieces of 1 Byte each, in order to work directly at byte level, working on 4 Bytes per time, speeding up the inversion process. In the following image we can see a more clear representation of the subdivision:

[Instantiation Porting]

The same concept has been applied to the output port m00_axis_tdata, where we merged 4 pieces of 8 bits each into 1 of 32 bits.
